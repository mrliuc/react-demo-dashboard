# 安装

### `cnpm install`

# 运行

启动后端服务

### `node server`

启动前端网页

### `npm start`

# 说明

存储数据的本地文件

`/data/data.json`

数据格式
	
    {
		"DashboardName": "bb",
		"WidgetName": "Red",
		"Name": "2",
		"GroupName": "2",
		"DataURL": "http://127.0.0.1:10001/dataSource1"
	}
